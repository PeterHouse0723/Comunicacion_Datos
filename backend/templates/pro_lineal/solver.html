¿Cómo funciona la Calculadora de Programación Lineal?

La Programación Lineal (PL) es una técnica matemática para optimizar (maximizar o minimizar) una función objetivo lineal sujeta a un conjunto de restricciones lineales. Se aplica en logística, producción, finanzas, redes y muchos problemas de toma de decisiones.

Función objetivo
Expresión lineal que se quiere maximizar o minimizar. Ejemplo: Z = 3x₁ + 5x₂. En la calculadora se ingresan solo los coeficientes: 3 5.
Restricciones
Desigualdades o igualdades que limitan las variables. Cada una en una línea con formato coef1 coef2 … op RHS. Operadores: <=, >=, =.
Variables de decisión
Las incógnitas del problema (x₁, x₂, …). Se asume que son ≥ 0 (no negatividad). La cantidad de variables la determina el número de coeficientes en la función objetivo.
Solución óptima
El punto dentro de la región factible donde la función objetivo alcanza su valor máximo o mínimo. La calculadora muestra cada iteración del método elegido.
Simplex
Método algebraico clásico. Ideal para problemas con restricciones ≤ y variables no negativas. Parte de una solución básica factible y mejora iterativamente hasta el óptimo. Recomendado para la mayoría de problemas.
Dos Fases
Para problemas con restricciones ≥ o =. Fase 1: minimiza variables artificiales para encontrar una solución factible. Fase 2: optimiza la función objetivo real. Muestra ambas fases en detalle.
Gráfico
Solo para 2 variables. Dibuja la región factible, los vértices y el punto óptimo. Excelente para entender visualmente el problema antes de usar métodos algebraicos.
Dualidad
Análisis primal–dual. Resuelve simultáneamente el problema original (primal) y su dual. Útil para análisis de sensibilidad: muestra precios sombra y holguras de las restricciones.
